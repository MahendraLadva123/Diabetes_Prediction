# Diabetes Prediction

This project uses machine learning to predict diabetes based on patient health attributes.

## Features
- Web interface (Flask) for predictions
- Machine learning model trained on the PIMA diabetes dataset
- Exploratory Data Analysis (EDA) notebooks

## Installation
1. Clone the repository:https://github.com/MahendraLadva123/Diabetes_Prediction
2. Install dependencies:  pip install -r requirements.txt
3. Run the Flask app:python app.py

4. Open `http://127.0.0.1:5000/` in your browser.

## Folder Structure
- `app.py`: Flask web application
- `diabetes.pkl`: Trained model
- `templates/`: HTML templates
- `notebooks/`: Jupyter notebooks for EDA and model training
- `requirements.txt`: Dependencies

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas, NumPy
- Bootstrap (Frontend)
